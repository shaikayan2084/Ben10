# Ben 10 Interactive Encyclopedia

A responsive Ben 10 fan encyclopedia featuring 25 Omnitrix aliens from across multiple Ben 10 series. Users can explore alien species, abilities, origins, and detailed information through an interactive web interface.

## Live Demo

https://shaikayan2084.github.io/Ben10/

## Features

- Interactive alien encyclopedia
- 25 Omnitrix aliens
- Organized by series
  - Classic Series
  - Alien Force
  - Ultimate Alien
  - Omniverse
- Responsive design
- Modern UI
- Smooth navigation
- Detailed alien information cards

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages
